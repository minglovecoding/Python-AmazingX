## Python Data Types

### **Numbers**

Four numeric types: integer, boolean, float, and complex.

- int (e.g., 1)
- bool (True/False)
- float (e.g., 1.23)
- complex (e.g., 1 + 2j)

### **String**

- Single `'` and double `"` quotes are the same
- Triple quotes allow multi-line strings
- Backslash `\` escapes characters
- Prefix `r` to prevent escapes
- Use slicing: `str[start:end:step]`

```
word = 'string'
sentence = "This is a sentence."
paragraph = """This is a paragraph,
spanning multiple lines."""
```

    elif guess > answer:
        print("Too high")
    else:
        print("Congratulations! You guessed it!")
