### 📌 code 3：判断是否为正数

```python
num = int(input("请输入一个整数："))
if num > 0:
    print("这是一个正数")
elif num < 0:
    print("这是一个负数")
else:
    print("这是零")
```

### 📌 code 5：猜数字小游戏

```python
import random
answer = random.randint(1, 100)
guess = -1
while guess != answer:
    guess = int(input("猜一个 1 到 100 之间的数字："))
    if guess < answer:
        print("猜小了")
    elif guess > answer:
        print("猜大了")
    else:
        print("恭喜你，猜对了！")
```
